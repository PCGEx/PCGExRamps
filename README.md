# PCGEx Ramps

**Inline ramp/curve authoring for PCG.**  
Encodes a curve or LUT into a single compact-text string, builds a transient UCurveFloat from it, and exposes that curve via soft object path so native PCG nodes can consume an inline-authored curve without a saved asset. It creates a transient UCurveFloat as a managed resource.
